Muat dan ikuti skill fable-auto.

Tugas pengguna:

$ARGUMENTS

Pilih sendiri apakah tugas dikerjakan langsung atau menggunakan
fable-luna, fable-sol, atau fable-terra.

Jangan meminta pengguna memilih agent.
Jangan menggunakan subagent apabila pekerjaan sederhana dapat diselesaikan
langsung dengan lebih cepat.

## Senior Engineer Mode Integration

Untuk tugas engineering nontrivial, muat dan ikuti skill
`senior-engineer-auto`.

Gunakan:

- Peta Auto untuk konteks proyek
- Senior Engineer Auto untuk skill, gate, dan evidence
- Fable Auto untuk pemilihan Luna, Sol, atau Terra

Jangan mengaktifkan Senior Engineer Mode untuk pertanyaan sederhana atau
command terminal singkat.

## Workflow Control Integration

Sebelum melakukan routing:

1. Jika kebutuhan atau keputusan belum jelas, gunakan skill `grilling`. Ajukan
   satu pertanyaan dengan rekomendasi lalu tunggu jawaban pengguna.

2. Jika Plan Mode aktif, jangan memanggil Luna, Sol, Terra, `Agent`, atau
   subagent untuk melakukan perubahan. Tetap read-only dan gunakan
   `create_plan` atau `update_plan`.

3. Untuk pekerjaan multi-langkah di luar Plan Mode, gunakan `todowrite`.
   Pertahankan tepat satu todo `in_progress` dan perbarui setiap penyelesaian.

4. Setelah rencana disetujui, ubah langkah rencana menjadi todo lalu gunakan
   Fable Auto untuk eksplorasi, implementasi, verifikasi, dan review.

5. Terra hanya boleh melakukan implementasi ketika Plan Mode tidak aktif.

6. Untuk tugas sederhana, jawab atau kerjakan langsung.
